このプログラムはfloodgate形式の棋譜を技巧の学習形式（2chkihu.csa2形式）に変換するためのプログラムです。
先手と後手のレートが2800以上でなければ変換して書き出しません.

正直言って自分用にテキトーに書きました。
（使用方法）
fg2gdb [folder_name]

[foldername]で指定されたフォルダ内の*.csaファイルを再帰的に探します。

https://github.com/daruma3940/fg2gdb/pull/3
棋譜番号部分に文字列が入ってしまう関係で技巧２ではうまく動かないといったことがあるそうです。
これはmain.cpp内のwrite_game_info関数でGD.eventを"0"に置き換えていただけるとうまく動くそうです。
